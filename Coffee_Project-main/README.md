# Bean There (App Folder)

This is the runnable Next.js app folder for Bean There.

For judge-facing project context and demo narrative, see the root `README.md`.

## Run

```bash
npm install
npm run dev
```

## Required `.env.local`

```env
NEXT_PUBLIC_AI_MODE=real
GEMINI_API_KEY=...
GEMINI_MODEL=gemini-2.5-flash
NEXT_PUBLIC_ENABLE_PLACES=true
GOOGLE_PLACES_API_KEY=...
```
